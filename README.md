## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- morgan (optional)
## Getting Started

- Install all dependencies (except for the optional ones) (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
